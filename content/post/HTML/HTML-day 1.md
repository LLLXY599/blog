# （一）、准备开发环境
	1.vscode（编写代码）+Google浏览器（看效果）
	2.插件：open in browser(打开网页)
	3.把Google浏览器设置为默认浏览器

# （二）、标签语法
	1.定义：超文本（链接）标记（标签<>）语言
	2.注：用（）代替<>



# （三）、基本骨架
	1.快速生成骨架：！
	2.网页名字：title
	3.整个网页：html
	4.网页头部：head
	5.网页主体：body

# （四）、标签关系
	1.父子关系（嵌套关系）
	2.兄弟关系（并列关系）

# （五）、文字标签
	1.ctrl + / ：注释捷键
	2.标题标签：h1至h6（双标签），独占一行，逐渐减小字体
		注：h1在一个网页中只用一次，其他的没有次数限制
	(strong)文字(/srong)：加粗
	3.b：加粗
		注：vscode中手动换行没用
	4.br：换行
	5.hr：水平线
	6.em / i：倾斜
	7.ins / u：下划线
	8.del / s：删除线

# （六）、图像标签及属性
	1.(img src="图片的url")
	2.alt：替换文本，图片无法显示的时候显示的文字
	3.title：鼠标悬停在图片上的时候显示的文字
	4.width：宽度
	5.height：高度

# （七）、超链接标签及属性
	1.（a  href=" 链接")文字（/a)
	2.href：写#代表空链接
	3.target：打开新窗口跳转

# (八)、音频标签及属性
	1.(audio src=" url ")(/audio)
	2.cotrols：显示音频控制面板，属性名与属性值一样简写
	3.loop：循环播放
	4.autoplay：自动播放，禁用

# （九）视频标签及属性
	1.(video src=" url ")(/video)
	2.cotrols：显示音频控制面板，属性名与属性值一样简写
	3.loop：循环播放
	4.autoplay：自动播放，禁用
	5.muted：静音播放

# （十）、综合案例-个人简介
	思路：从上到下，先整体再局部
